# NoteBean Landing Page

This is a React-based landing page for the NoteBean application, built with Vite and Vanilla CSS.

## Project Structure

- `src/components/`
  - `layout/`: Global layout components like `Navbar`, `Footer`, and `Layout` wrapper.
  - `sections/`: Specific sections for the landing page (Hero, Features, etc.).
  - `ui/`: Reusable primitive components (Buttons, Inputs, etc.).
- `src/pages/`: Main page components.
- `src/styles/`: CSS files.
  - `variables.css`: Design tokens and CSS variables.
  - `global.css`: Global resets and typography.
- `src/assets/`: Static assets like images and icons.
- `src/hooks/`: Custom React hooks.
- `src/utils/`: Utility functions.

## Implementation Details

- **Framework**: React 18
- **Build Tool**: Vite
- **Routing**: React Router DOM v6
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (CSS Variables for theme/tokens)

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
